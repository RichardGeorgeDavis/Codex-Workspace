# Wiki Starter Pages

This folder holds starter content for the optional GitHub wiki for Codex Workspace.

The intended model is:

- keep the repository `README.md` and `docs/` folder as the canonical documentation surface
- keep the GitHub wiki short and navigational
- link back to tracked docs for durable technical detail

Suggested wiki pages:

- `Home.md`
- `_Sidebar.md`
- `Workspace-Hub.md`
- `Repo-Manifests.md`
- `FAQ.md`

Paste these into the GitHub wiki after enabling the wiki feature in the repository settings.
